# goal

## Setup

```sh
bun install
```

## Testing

Run tests for packages:

```sh
bun run test:packages
```

Run tests for applications:

```sh
bun run test:apps
```

Run all tests:

```sh
bun run test:all
```

## Linting

```sh
bun run biome:fix
```

## Running:

```sh
bun run dev
```

open <http://localhost:3000>

## Structure

Workspace packages live under:

- `apps/*`
- `packages/*`
