# goal

To install dependencies:

```sh
bun install
```

To run tests:

```sh
bun run test:packages
bun run test:apps
bun run test:all
```

To lint and format the codebase:

```sh
bun run biome:fix
```

To run:

```sh
bun run dev
```

open <http://localhost:3000>

Workspace packages live under:

- `apps/*`
- `packages/*`
